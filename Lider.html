1️⃣ Primeiro: Existe obrigação formal?

Depende de três fatores:

✅ Existe contrato?

Se houver contrato com cláusula de disponibilidade → é obrigação.

Se não houver contrato → é acordo verbal (mais frágil).

✅ Existe calendário pré-definido?

Se os dançarinos já sabiam que março é mês de shows → é compromisso.

Se é totalmente surpresa → vira negociação.

✅ É grupo profissional ou colaborativo?

Grupo profissional → compromisso com antecedência + prontidão.

Grupo informal → depende de disponibilidade.
